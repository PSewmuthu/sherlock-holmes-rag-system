# Sherlock Holmes RAG System 🕵️‍♂️🔍

The **Sherlock Holmes RAG System** is a sophisticated AI-driven platform designed to act as a digital companion to the Great Detective’s adventures. By bridging the gap between static literary archives and Large Language Models, this system allows users to engage in deep, factual conversations about the 62 original stories, character histories, and the Victorian world of 221B Baker Street.

## 🚀 Overview

This project implements a complete **Retrieval-Augmented Generation (RAG)** pipeline. Unlike standard AI models that may "hallucinate" details about classic literature, this system anchors every response in the verified text of the _Arthur Conan Doyle Encyclopedia_. It treats the entire Sherlockian canon as a structured knowledge base, ensuring that every deduction is supported by textual evidence.

## 🌟 Key Features

- **Canonical Accuracy**: Provides precise information on the "62 stories" (56 short stories and 4 novels) written by Sir Arthur Conan Doyle.
- **Contextual Intelligence**: Maintains the thread of complex literary discussions, from the chemistry of Holmes's experiments to the geography of London.
- **Source-Backed Responses**: Every answer generated by the system is retrieved from a high-fidelity vector database of processed encyclopedia entries.
- **Deductive Reasoning**: Optimized prompts guide the AI to mirror the analytical and professional tone of the Sherlock Holmes series.

## 🏗️ System Architecture

The system follows a modern AI workflow to transform web-based literature into a conversational intelligence:

1.  **Knowledge Ingestion**: The system systematically maps the target encyclopedia, identifying and capturing every relevant story and biographical entry.
2.  **Semantic Indexing**: Raw data is cleaned and converted into high-dimensional vector embeddings, creating a mathematical map of the Holmesian universe.
3.  **Vector Retrieval**: When a user asks a question, the system performs a similarity search to find the exact "clues" or paragraphs needed to answer.
4.  **Augmented Generation**: The retrieved context is passed to the Language Model, which synthesizes a natural, accurate response.

## 🛠️ Tech Stack

- **Orchestration**: [LangChain](https://python.langchain.com/)
- **Vector Database**: [ChromaDB](https://www.trychroma.com/)
- **Data Processing**: Python-based HTML parsing and text normalization.
- **Intelligence Layer**: Gemini API.

## 🏁 Getting Started

### Installation

1.  **Clone the Repository**:

    ```bash
    git clone [https://github.com/pasindusewmuthu/sherlock-holmes-rag-system.git](https://github.com/pasindusewmuthu/sherlock-holmes-rag-system.git)
    cd sherlock-holmes-rag-system
    ```

2.  **Install Dependencies**:

    ```bash
    pip install -r requirements.txt
    ```

3.  **Start the Detective**:
    ```bash
    uv run fastapi run main.py
    ```
